# スプレッドシート仕様

保存先シートは`看護介護記録`を優先し、存在しない場合は`Records`を使用します。

OTA取込用の列は、次の順序・表記に統一します。

| 列 | ヘッダー | 保存値 |
|---|---|---|
| A | 看護介護記録氏名 | GAS側の利用者マスタから再取得した正式名称 |
| B | 看護介護記録日付 | 日本時間の作業日、または確認画面で職員が変更した日付 |
| C | 看護記録者 | GAS側の職員マスタから再取得した正式名称 |
| D | 記録血圧 | 収縮期と拡張期の両方がある場合のみ`120~70`形式 |
| E | 記録脈 | 脈拍 |
| F | 看護介護記録 | 検証済みの整形済み記録本文 |

血圧は片方だけの場合、自動確定せず保存時にもエラーにします。
